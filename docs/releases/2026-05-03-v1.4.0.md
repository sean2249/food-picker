# Food Picker Release Notes — v1.4.0

**發布日期：** 2026-05-03

---

## 本次更新重點

### 1. 全新視覺語言：日系手帳・粉圓字
整個 App 換上一套更柔軟、更像手寫小本子的視覺。字體從中文襯線（Noto Serif TC）改為 **jf-openhuninn 粉圓 2.1**，圓潤的筆觸讓畫面溫度更高、更不像科技產品。
背景換成米白亞麻色，疊上一層淡雜誌紙紋與顆粒底紋；每頁內容區域多了一層柔光紙張感襯底，閱讀時不會與紋理打架。

### 2. 新的品牌符號：提拉米蘇
首頁的主視覺從原本的飯糰換成手繪的**提拉米蘇蛋糕**——可可粉、馬斯卡彭、咖啡浸海綿三層分明，加上一顆咖啡豆裝飾與三顆閃爍星星。
左上角導覽列也帶有一個迷你提拉米蘇條紋方塊作為品牌印記，無論在哪一頁都能感受到一致的識別。

### 3. 響應式導覽列：桌機展開、手機漢堡
**手機（<640px）**：右上角是漢堡按鈕，點開後會緩緩滑下選單面板，三條筆刷風格的線條會自然旋轉成 ×。
**桌機（≥640px）**：直接顯示完整連結列，主要動作（推薦給我）以填色橘色膠囊突顯，其他連結為幽靈樣式。
這個改動讓手機操作不擁擠，桌機使用又不必多按一次選單。

### 4. 全部 emoji 換成手繪線條圖示
原本卡片上的 🦶 🚌 🚗 🚄 ✈️ 與 📍 表情符號，全部替換為**手繪風格的 SVG 線條圖示**。圖示的筆觸與字體呼應，視覺上更一致；走路、公車、汽車、火車、飛機，每一種距離都有獨立的小插畫。
評分星星、麵碗、書籤點等 UI 圖示也改採同一套筆觸語言。

### 5. 按下會像布丁的主要按鈕
所有主要動作按鈕（首頁推薦、推薦給我、就選這家、新增餐廳）採用堆疊式陰影製造立體感，按下時陰影壓扁、按鈕往下沉，並帶有一段約 360ms 的「布丁彈跳」動畫。
只有按下時才會彈跳，滑鼠經過不會無意義地跳動。

### 6. 「推薦給我」頁面整理
- 搜尋輸入框放大、加上日記式說明：「想吃什麼？也可以說說心情」
- 篩選條件改為可折疊抽屜，標題會顯示目前生效的篩選數量
- 篩選器內的選項改為丸藥式（pill）按鈕：選中為橘色淡底，未選則為灰邊框
- 距離滑桿改採品牌橘色高亮
- 推薦中時，骰子圖示會輕快旋轉，按鈕文字變成「正在挑…」
- AI 推薦理由改成有「挑選筆記」標題的虛線卡片，每家餐廳上方有編號注解

### 7. 「餐廳清單」頁面整理
- 標題旁多了書籤點圖示與「翻翻你蒐藏的小店本」短句
- 「全部 / 未造訪 / 已造訪」分頁加上每個分類的數量
- 標題列右側新增了快速「+ 新增」按鈕
- 沒有資料時，顯示一張虛線邊框的空白卡，第一次使用會邀請你「記下第一家」

### 8. 「新增 / 編輯餐廳」表單重設計
- 每個欄位前都帶有「・」起手符與淺灰小字說明，視覺像手寫表單
- 「已造訪」改為兩顆切換丸藥（**還沒去** / **吃過了**），不再是不顯眼的勾選框
- 評分從 1–5 數字按鈕換成**五顆星**：選中亮起品牌色，旁邊有清除選項
- 「建議標籤」按鈕加上虛線邊框與閃爍星星圖示，產生中時星星會旋轉
- 編輯頁的 AI 摘要改為虛線卡片，視覺上更貼近「貼進筆記本」的感覺

### 9. 餐廳卡片內容微調
- 「已訪」徽章從硬邊框旋轉貼紙，改為淡橘色虛線圓章，旋轉角度更小、更柔和
- 短評加上中文引號 「 」 包夾，讀起來更像真人筆記
- 標籤從滿色橘色 badge 改為灰底細邊條，多個標籤一起顯示時不再眼花
- 「就選這家」按鈕改為麵碗圖示 + 布丁式按壓互動

## 移除功能

- **lucide-react 圖示與 emoji 表情：** 為了讓全 App 視覺一致，所有功能圖示（距離、星星、勾選、加號等）都改為手繪 SVG 取代。
- **Noto Serif TC 中文字體：** 改為更貼近手帳氛圍的 jf-openhuninn 粉圓 2.1。

## 使用流程（新版）

1. 開啟 App：首頁正中央是會閃爍的提拉米蘇插畫，下方是兩顆按鈕——主要的「我餓了！推薦我」與次要的「新增一間餐廳」。
2. 想吃飯時：按主要按鈕（會像布丁一樣彈跳）→ 進到「推薦給我」。
3. 在輸入框寫下想吃什麼或心情；如要更精準，展開「篩選條件」抽屜，調整造訪狀態、標籤、距離上限。
4. 按下橘色按鈕開始推薦：骰子圖示會旋轉，過幾秒得到三家建議與一段「挑選筆記」。
5. 看中哪家就按該卡的「就選這家」——畫面下方會浮現「已記錄你的選擇」的細線小印記。
6. 平常想翻紀錄時：手機點右上角漢堡 →「餐廳清單」；桌機直接點上方連結。清單頁可用標籤切換造訪狀態，或從右上角「+ 新增」開始記新店。
