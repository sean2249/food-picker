# Food Picker Release Notes — v1.0.0

**發布日期：** 2026-04-20

---

## 本次更新重點

### 1. AI 摘要
新增「生成 AI 摘要」按鈕，點一下即可根據品項、標籤與短評，自動生成一句描述餐廳特色的中文摘要。
摘要可在送出前手動調整，儲存後會顯示在餐廳卡片的名稱下方，讓你一眼掌握每家店的個性。

### 2. 餐廳編輯
餐廳清單每張卡片新增「編輯」按鈕，可直接修改已存入的餐廳資料（名稱、捷運站、品項、距離、標籤、評分、短評、AI 摘要）。
再也不用刪掉重填——造訪之後補上評分與心得，或更新菜單品項，都更方便了。

### 3. 標籤管理
新增餐廳或編輯時，AI 生成的標籤現在可以個別移除（點 ✕），也可以直接在輸入框打字後按 Enter 手動新增自訂標籤。
標籤完全由你掌控，不再是只能整批替換的唯讀結果。

### 4. 距離用語化
距離欄位從數字格子改為滑桿，並附上對應的中文說明，例如「走路 5 分鐘」、「搭車快一小時」、「開車才方便」。
選完就知道這家店大概有多遠，不用再自己換算數字的含義。

### 5. 餐廳清單篩選
餐廳清單頂部新增三個分頁：**全部 / 未造訪 / 已造訪**。
快速找到還沒去過的新目標，或回顧已經打卡的老朋友。

### 6. 推薦篩選條件
推薦頁面新增可展開的「篩選條件」區塊，讓你在推薦前先設定：
- **已造訪**：全部、只推未造訪、只推已造訪
- **標籤**：選取特定標籤（不選代表不限）
- **距離上限**：設定最遠可接受的距離

三個條件可自由組合，系統只會從符合條件的餐廳中挑選推薦結果。

---

## 使用流程（新版）

```
新增餐廳
  → 填寫名稱、捷運站、品項、距離滑桿
  → 勾選「已造訪」後填入日期、評分、短評
  → 點「建議標籤」自動生成，或手動輸入標籤
  → 點「生成 AI 摘要」產生一句特色描述
  → 送出儲存

編輯餐廳
  → 餐廳清單 → 點「編輯」→ 修改任意欄位 → 儲存變更

篩選清單
  → 餐廳清單頂部選分頁：全部 / 未造訪 / 已造訪

推薦時篩選
  → 推薦頁 → 展開「篩選條件」→ 設定造訪狀態、標籤、距離 → 點「推薦給我！」
```
