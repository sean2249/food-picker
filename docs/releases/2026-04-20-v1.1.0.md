# Food Picker Release Notes — v1.1.0

**發布日期：** 2026-04-20

---

## 本次更新重點

### 1. 造訪狀態視覺標示
餐廳卡片現在以顏色區分已造訪與未造訪：綠色「✓ 已造訪」代表你去過，灰色「未造訪」代表還沒去。一眼就能知道哪些餐廳等著你探索。

### 2. 表單新增取消按鈕
編輯餐廳時，底部新增了「取消」按鈕，點擊後直接返回上一頁，不需要按瀏覽器上一頁或重新整理。

### 3. 推薦篩選器保持開啟 + 重新推薦
收到推薦結果後，篩選器會自動保持開啟狀態，方便你調整條件後直接按「調整篩選後重新推薦」，不需要重新展開篩選器再操作。

### 4. AI 推薦更精準：優先參考摘要與標籤
推薦引擎現在優先考慮餐廳的 AI 摘要與標籤（原本是評分與捷運站），讓推薦結果更貼近餐廳的實際特色與你的心情。

### 5. 首頁更新記錄
首頁新增「更新記錄」區塊，可展開查看每個版本的新功能，讓你隨時掌握 App 最新變化。

---

## 移除功能

- **造訪日期欄位：** 新增／編輯餐廳時不再需要填寫造訪日期，簡化表單操作。

---

## 使用流程（新版）

```
推薦流程：
1. 前往「推薦」頁面，展開篩選器設定條件
2. 按「推薦我！」取得結果
3. 篩選器保持展開 → 直接調整條件
4. 按「調整篩選後重新推薦」取得新結果

編輯餐廳：
1. 在餐廳列表找到要編輯的餐廳
2. 點擊「編輯」進入編輯頁面
3. 修改完成按「儲存變更」，或按「取消」返回
```
