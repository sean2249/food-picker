# Food Picker Release Notes — v1.3.0

**發布日期：** 2026-04-22

---

## 本次更新重點

### 1. 全新視覺風格：溫暖手帳感設計 (暖橘色（#E46C0A), 米白, Noto Serif TC)
整個 App 的配色從冷灰白切換為暖色調——背景是帶奶油感的米白，主要動作按鈕維持溫暖橘色（#E46C0A），文字改為暖棕色而非純黑。
字體換成 **Noto Serif TC**，搭配中文襯線的筆觸，讓介面更像一本親手書寫的美食筆記，而不是科技產品。

### 2. 手繪食物插圖背景
畫面背景加入淡淡的手繪線稿圖案——麵碗、筷子、餃子、香草葉、叉子等食物輪廓，以低對比度浮現在頁面之後。
這層紋理不搶眼，但讓整個操作環境更有生活感與溫度。

### 3. 餐廳卡片全面重設計
距離標示改為直覺的表情圖示：步行距離顯示 🦶、搭車距離顯示 🚌、開車距離顯示 🚗，一眼就能判斷遠近。
已造訪的餐廳在卡片右上角加上「已訪」圓形印章，視覺上更像蓋了個章的旅遊筆記。

### 4. 刪除確認移入卡片內，不再跳出瀏覽器彈窗
點擊「刪除」後，確認按鈕直接出現在卡片底部（顯示「確定？是 / 否」），取代原本打斷體驗的瀏覽器 `confirm()` 對話框。
整個流程維持在畫面內，操作更順暢，也更符合 App 的整體視覺風格。

### 5. 餐廳清單載入時有骨架動畫: 在讀取資料期間，現在會顯示卡片形狀的灰色骨架動畫（而非空白畫面）
餐廳清單在讀取資料期間，現在會顯示卡片形狀的灰色骨架動畫（而非空白畫面）。
這讓等待過程更有反饋感，不會讓人以為畫面當掉了。

### 6. 無障礙體驗改善
距離滑桿加上螢幕閱讀器標籤、評分按鈕的觸控區域放大至 44px、標籤刪除按鈕加大點擊範圍。
使用篩選器時，造訪狀態選項現在有正確的語意群組，對輔助工具更友善。

---

## 使用流程（新版）

1. 開啟 App，感受溫暖的米白底色與手繪食物背景紋理。
2. 進入「餐廳清單」，等待時有骨架動畫提示載入中。
3. 查看各卡片——距離以 🦶 / 🚌 / 🚗 圖示標示，已造訪的餐廳卡片右上角有「已訪」印章。
4. 點擊「刪除」後，確認選項直接出現在卡片內，選「是」才執行刪除。
5. 前往「推薦給我」，造訪狀態篩選器語意更清晰，距離滑桿對輔助工具更友善。
